# 💸 Splitwise Clone – Neurix Full-Stack SDE Intern Assignment

This is a simplified Splitwise clone that lets users track shared group expenses, calculate balances, and determine who owes whom. Built with **FastAPI + PostgreSQL (backend)** and **React + TailwindCSS (frontend)**.
